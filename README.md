#### 一、说明

- 这是我的博客附属的刷题记载分站 
- 1、全部使用`C++11`语法（部分由于无法支持C++11语法的，会直接使用C++98语法）
- 2、代码会修改，尝试精简
- 3、追求“良好的代码风格” ，再考虑精简性
- 4、不保证效率最高性  
- 5、编译环境：`linux下g++`
